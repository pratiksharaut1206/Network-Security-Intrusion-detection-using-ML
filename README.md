# Network Intrusion Detection ML Project

## 📌 Overview
This project predicts whether a network session is malicious using machine learning.

## 🚀 Features
- Data preprocessing & EDA
- Feature engineering
- Random Forest model
- Pipeline implementation
- Streamlit deployment

## 📂 Files
- dataset.csv
- model.pkl
- app.py
- notebook.ipynb

## ▶️ Run App
streamlit run app.py

## 🧠 Model
Random Forest Classifier with feature engineering.

## 📊 Evaluation
Metrics used:
- Accuracy
- Precision
- Recall
- F1-score
